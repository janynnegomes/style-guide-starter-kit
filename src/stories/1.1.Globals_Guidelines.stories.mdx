# Guidelines


This project uses [Sass](https://sass-lang.com) as a preprocessor for CSS.

<div className="subheading">DOs and DON'Ts</div>

* DO single-line comments for comments that other developers need to see, since they won't get compiled into CSS final stylesheet.
* DON'T use inline styles
* DO follow hyphen-separated class name. <code>E.g. list-block</code> 
 Modifier classes use a double-hyphen naming convention. <code>E.g. list-block--profile</code>
* DO browse through the variables, globals, spacing and typography files, they contains plenty reusable code.
