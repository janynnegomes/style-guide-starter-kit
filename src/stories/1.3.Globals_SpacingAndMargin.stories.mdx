# Spacings

<div className="tip-wrapper">
  <span className="tip">Reference files:</span><code>assets/_variables/spacing.scss</code>,<code>assets/partials/spacing.scss</code>
</div>

Use variables or classes for spacing.

Spacing helpers can be used for padding, margins, posicioning elements and etc.

The spacing variables are <code>$spacing</code>, <code>$spacing-1</code>,<code>$spacing-2</code> and <code>$spacing-half</code>.

The default <code>$spacing</code> variables's value is 10 pixels.
