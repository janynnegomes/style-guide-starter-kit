# Media Queries



The application primarily uses min-width media query variables since the layout is mobile-first. However max-width media query variables are also available.


## Breakpoint functions

<div className="tip-wrapper">
  <span className="tip">Reference file:</span><code>assets/functions/breakpoints.scss</code>
</div>

You can use the <code>breakpoint</code> function to provide necessary media-query range for each type of device size.

The breakpoint parameters are: <code>mobile</code>,<code>tablet</code> and <code>desktop</code>

```
<style>
.selector{
  
  width:100%;

  @include breakpoint(mobile) {
      
    width:60%;

  }

  @include breakpoint(tablet) {

    width:50%;

  }

  @include breakpoint(desktop) {
      
    width:60%;

  }

 
}
</style>

```


## Media Queries

<div className="tip-wrapper">
  <span className="tip">Reference files:</span><code>assets/_variables/media-queries.scss</code>
</div>

Min-width media query variables are: <code>$media-sm</code>,<code>$media-md</code>,<code>$media-lg</code>

Max-width media query variables are: <code>$media-sm-max</code>,<code>$media-md-max</code>,<code>$media-lg-max</code>

Media queries are usually nested within the selector they are associated with.




```
<style>
.selector{
  
  width:100%;

  // Media queries: Min Width
  @media (min-width: $media-sm) {
      
    width:60%;

  }

  @media (min-width: $media-md) {
      
    width:50%;

  }

  @media (min-width: $media-lg) {
      
    width:90%;

  }

  // Media queries: Max Width
  @media (max-width: $media-sm-max) {
      
    width:60%;

  }

  @media (max-width: $media-md-max) {
      
    width:50%;

  }

  @media (max-width: $media-lg-max) {
      
    width:90%;

  }
}
</style>

```
